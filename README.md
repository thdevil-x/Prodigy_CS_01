# Caesar Cipher

## Description

This project implements the Caesar Cipher algorithm using Python.

The program allows users to:

- Encrypt messages
- Decrypt messages
- Enter custom shift values
- Preserve spaces and symbols

## Features

- Encrypt text
- Decrypt text
- User-friendly menu
- Supports uppercase and lowercase letters

## Technologies Used

- Python 3

## How to Run

```bash
python caesar_cipher.py
```

## Example

Message:

```
HELLO
```

Shift:

```
3
```

Encrypted:

```
KHOOR
```

Decrypted:

```
HELLO
```
